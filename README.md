## About
Fine-tuning is a method of modifying generalized LLMs to work to have better performance in specific domains.


## Learning Objectives

*You will be able to:*
- Describe fine-tuning use cases.
- Explain the goals and tradeoffs of model fine-tuning and methods (LoRA, QLoRA, etc.).
- Apply Low-Rank Adaption (LoRA) of LLMs.
